# HTML Tower Defense

A simple tower defense project made with plain **HTML, CSS, and JavaScript**. The goal of the project is to experiment with building a strategy-style game that runs directly in the browser without needing any external frameworks or engines.

The game focuses on a fantasy theme where the player selects a map and a hero before starting waves of enemies.

---

## About the Project

This started as a small project to test how far a browser-based tower defense game could be pushed using just basic web technologies. Instead of using something heavy like Unity or a full game engine, everything is built with standard web tools.

The idea is to keep the game lightweight and easy to run while still having a polished interface and responsive controls.

---

## Current Features

* Map selection system
* Hero selection menu
* Fantasy-themed interface
* Animated UI elements
* Basic game setup screen
* Designed to run directly in the browser

---

## Running the Project

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/html-tower-defense.git
```

2. Open the folder

3. Launch `index.html` in your browser

That's it. No setup or installs needed.
